# Oblig3_software

Jeg startet med å lage en ny repository i github som jeg kalte Oblig3_software, og koblet den til med pycharm sånn at jeg kan bruke push og pull i GitKraken.

Etter noen feil tester ble alle testene godkjent til slutt.
